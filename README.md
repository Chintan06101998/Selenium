# SeleniumDemo

# Description

- This is the demo project for selenium begineers to start with the basic
- It covers all of the important part of Automation took Selenium
- Accessor
- Assert
- Verify
- Wait cases
- Valid and Invalid Credential for Login
- Test Recording
- Debugging
- Variable
- data storing
- Reusable funcation

- Get Commands
- Navigation Commands
- Locator Commands
- Browser Commands

- Foor loop
- Do-while loop
- Time loop
